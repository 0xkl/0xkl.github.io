# col 命令
***过滤控制字符***

**补充说明**

**col命令**是一个标准输入文本过滤器，它从标记输入设备读取文本内容，并把内容显示到标注输出设备。在许多UNIX说明文件里，都有RLF控制字符。当我们运用Shell特殊字符">"和">>"，把文件的内容输出成纯文本文件时，控制字会变成乱码，col指令则能有效滤除这些控制字符。

### 语法
```bash
col [选项]...
```

### 选项
```
 -b, --no-backspaces    过滤掉所有的控制字符，包括RLF和HRLF
 -f, --fine             过滤除RLF字符，但允许将HRLF字符呈现出来
 -p, --pass             传递未知控制序列
 -h, --tabs             将空格转换为制表符
 -x, --spaces           以多个空格字符来表示
```

### 参数
1> 过滤test.txt文件中的所有控制字符并标准输入到test1.txt文件
```bash
cat test.txt | col -v > test1.txt
```
